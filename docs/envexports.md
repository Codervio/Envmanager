# Exports and setenv variables

A variable can be prefix set with export or setenv.
Export is using on ksh, bash or bourne shells.
Setenv is using on CSH shells.

## Examples

Export sample:

```shell
export FOO1=foo1 #comment
```

Will return:

```shell
(string) 'foo1'
```

Setenv sample:

Export sample:

```shell
setenv FOO1=foo1 #comment
```
