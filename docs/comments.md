# Comments

Comments can be write on single line after variable or defined inline.

## Examples

A comment after variables set:

```shell
FOO1=foo1 #comment
export FOO1=foo1 #comment
setenv FOO1=foo1 #comment
```

As new line comment:

```shell
# comment
# a comment #comment
## A main comment ##
```
